# Quick Free CV Builder Launch Checklist

## Core checks
- [ ] Landing page loads correctly
- [ ] Create New CV starts a new draft
- [ ] Resume last draft works
- [ ] Upload Existing CV supports JSON / TXT / PDF / DOCX / image
- [ ] Builder edits persist in localStorage
- [ ] PDF export downloads correctly
- [ ] JSON export downloads correctly

## Content checks
- [ ] Personal info fields render in preview
- [ ] Experience renders correctly
- [ ] Education renders correctly
- [ ] Skills render correctly
- [ ] Projects render correctly
- [ ] Languages render correctly
- [ ] Optional sections render correctly
- [ ] Volunteer, interests, references are available

## Visual checks
- [ ] Classic template looks polished
- [ ] Modern template looks polished
- [ ] Compact template looks readable
- [ ] ATS mode looks clean and monochrome
- [ ] Mobile edit / preview toggle works
- [ ] Sidebar drag reorder works

## Export checks
- [ ] Long CV page breaks are acceptable
- [ ] Photos do not distort in PDF
- [ ] Optional sections do not break export
- [ ] Links remain readable

## Release checks
- [ ] README is current
- [ ] Favicon is present
- [ ] Meta title / description are correct
- [ ] Vercel / SPA redirects are configured
