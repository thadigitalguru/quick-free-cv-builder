# Quick Free CV Builder — Launch Note

Quick Free CV Builder is a free, privacy-first CV builder with no sign-up, no paywall, no watermark, and no backend dependency.

## What ships
- Landing page with trust-first messaging
- Two-pane builder workspace
- Autosave to localStorage
- Live CV preview
- PDF export
- JSON export/import
- PDF, DOCX, TXT, and image import support
- ATS mode plus Classic / Modern / Compact templates
- Optional sections for volunteer work, interests, awards, and references
- Profile photo upload with crop controls

## Deployment
The app is configured for SPA hosting and can be deployed on Vercel or Netlify with client-side routes enabled.
